I linguaggi usati

Per realizzare la WebApp abbiamo utilizzato diverse tecnologie, studiate in questi 3 anni: MySQL, JSON, PHP, JavaScript, HTML, CSS, Python

Il server

Il sito è ospitato gentilmente sui server di thage.me. Sulla macchina gira la distro Linux di Ubuntu, tenuta online da 2 GB di RAM e 2 processori.

Gli account

Per poter aggiungere elementi al nostro database è necessario accedere con una password e un username. Questi dati, ovviamente ritenuti sensibili, vengono salvati in un database MySQL e codificati con un hash variabile sino a 255 caratteri. Nessuna password viene salvata in chiaro o decodificata durante i processi di autenticazione.
